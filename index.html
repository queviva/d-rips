d-rips is a single-div of css that apparently drops random streaks of paint in any div with the <class="d‑rips">

mouse-over, or touch, the image to make it seeth with color - you might have to try more than once since the number and size of streaks may vary widely; as per yoosh, the color changes on scroll
set these css-var values in any element that uses the d‑rips class 

.d-rips {

    --drip-col:     // main color
    --hi-color:     // extra color || white
    --lo-color:     // extra color || black

    --drip-size:      // radius of a drip
    --drip-length:    // max length to streak
    --frame-width:    // width of the element
    --drip-duration:  // time dripping lasts

}

the ‑‑drip‑size and ‑‑drip‑length variables are simply starting points - the actual size and length of each drips varies ofcourse
