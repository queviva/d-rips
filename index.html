d-rips.css is a single-div of css that apparently drops random streaks of paint in any div with the <class="d‑rips">, but since that isn't possible, something else must be going on

mouse-over, or touch, the image to make it seeth with color - you might have to try more than once since the number and size of streaks varies wildly; as per yoosh, the colors change when scrolling over the image
include the style sheet, then set these css-var values in any element that uses the d‑rips class 

          
<link rel="stylesheet" href="d-rips.css">

<div id="sample" class="d-rips"> ... </div>

<style>

#sample {

    --drip-col:     // main color
    --hi-color:     // extra color || white
    --lo-color:     // extra color || black

    --drip-size:      // radius of a drip
    --drip-length:    // max length to streak
    --frame-width:    // width of the element
    --drip-duration:  // time dripping lasts

}
<style>
